# Vegetarian Recipes

Vegetarian Recipes è una web app sviluppata con React che permette agli utenti di cercare ricette vegetariane tramite l'API di Spoonacular.

## Funzionalità

- Ricerca di ricette vegetariane
- Visualizzazione immagini e titoli delle ricette
- Pagina dettaglio per ogni ricetta
- Ingredienti e istruzioni complete
- Responsive design
- UI semplice e intuitiva

## Tecnologie Utilizzate

- React
- React Router DOM
- Axios
- React Hooks
- Vite
- CSS3
- Spoonacular API
